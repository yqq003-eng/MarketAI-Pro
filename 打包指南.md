### 1. 安装 Node.js (v22+)

### 2. 安装 Visual Studio Build Tools（用于原生模块编译）
- 下载地址：<ADDRESS_REMOVED>
- 安装时勾选 **"使用C++的桌面开发"** 工作负载

### 3. 配置国内镜像（加速下载）

在命令提示符或PowerShell中执行：
```bash
# 配置npm镜像
npm config set registry https://registry.npmmirror.com

# 配置electron镜像
npm config set electron_mirror https://npmmirror.com/mirrors/electron/

# 配置electron-builder二进制镜像
npm config set electron_builder_binaries_mirror https://npmmirror.com/mirrors/electron-builder-binaries/

# 验证配置
npm config list
```

## 🚀 打包步骤（推荐方式）

### 方式一：使用已准备好的项目文件

1. **解压项目文件**
   - 将 `MarketAI-Pro-Complete.zip` 解压到任意目录（建议路径不含中文和空格）
   - 例如：`C:\MarketAI-Pro\`

2. **打开命令提示符（以管理员身份运行）**

3. **进入项目目录**
   ```bash
   cd C:\MarketAI-Pro
   ```

4. **安装依赖**
   ```bash
   # 安装根目录依赖（Electron等）
   npm install --legacy-peer-deps
   
   # 安装前端依赖
   cd src\frontend
   npm install --legacy-peer-deps
   cd ..\..
   ```

5. **构建前端**
   ```bash
   cd src\frontend
   npm run build
   cd ..\..
   ```

6. **打包成EXE**
   ```bash
   npx electron-builder --win --x64
   ```

7. **获取安装包**
   - 打包完成后，EXE安装文件在：`dist_electron\` 目录
   - 文件名类似：`MarketAI Pro Setup 1.0.0.exe`

## 🌐 方式三：使用 GitHub Actions 自动打包（推荐）

如果您有GitHub账号，可以使用GitHub Actions自动打包：

### 步骤：

1. **创建GitHub仓库**
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git remote add origin https://github.com/您的用户名/MarketAI-Pro.git
   git push -u origin main
   ```

2. **添加GitHub Actions配置文件**
   - 创建 `.github/workflows/package.yml`
   - 配置文件内容见下方

3. **推送代码到GitHub**
   ```bash
   git add .
   git commit -m "Add GitHub Actions workflow"
   git push
   ```

4. **下载打包好的EXE**
   - 进入GitHub仓库页面
   - 点击 "Actions" 标签
   - 选择最新的构建任务
   - 在 "Artifacts" 区域下载 `MarketAI-Pro-EXE`

### GitHub Actions 配置文件示例：

```yaml
name: Package Electron App

on:
  push:
    branches: [ main ]
  pull_request:
    branches: [ main ]

jobs:
  package-windows:
    runs-on: windows-latest
    
    steps:
    - uses: actions/checkout@v3
    
    - name: Setup Node.js
      uses: actions/setup-node@v3
      with:
        node-version: '22'
        
    - name: Configure npm mirror
      run: |
        npm config set registry https://registry.npmmirror.com
        npm config set electron_mirror https://npmmirror.com/mirrors/electron/
        npm config set electron_builder_binaries_mirror https://npmmirror.com/mirrors/electron-builder-binaries/
        
    - name: Install dependencies
      run: |
        npm install --legacy-peer-deps
        cd src/frontend
        npm install --legacy-peer-deps
        cd ../..
        
    - name: Build frontend
      run: |
        cd src/frontend
        npm run build
        cd ../..
        
    - name: Package Electron app
      run: |
        npx electron-builder --win --x64
        
    - name: Upload EXE artifact
      uses: actions/upload-artifact@v3
      with:
        name: MarketAI-Pro-EXE
        path: dist_electron/*.exe
```

## ⚠️ 常见问题解决

### 问题1：electron下载失败
**解决方案**：
```bash
# 手动下载electron放到缓存目录
# 1. 下载地址：<ADDRESS_REMOVED>
# 2. 将下载的文件放入：C:\Users\您的用户名\AppData\Local\electron\Cache\
# 3. 重新运行打包命令
```

### 问题2：sqlite3编译失败
**解决方案**（已在package.json中移除sqlite3依赖）：
- 如果使用数据库功能，可以：
  1. 使用预编译的sqlite3：`npm install sqlite3@5.1.7 --build-from-source=false`
  2. 或者改用其他数据库（如lowdb、nedb等纯JS数据库）

### 问题3：前端构建失败
**解决方案**：
```bash
# 检查vue.config.js是否有错误
# 检查src/frontend/package.json依赖是否完整
# 尝试删除node_modules后重新安装
```

### 问题4：electron-builder打包失败
**解决方案**：
```bash
# 清理缓存后重试
npx electron-builder clean
npm cache clean --force
rm -rf node_modules
rm -rf dist_electron
npm install
npx electron-builder --win --x64
```
