# 智能获客（个微）模块开发总结

## ✅ 已完成功能

### 📡 后端API路由 (wechat.js)

| API路径 | 方法 | 功能说明 |
|---------|------|----------|
| `/api/wechat/map-customers` | GET | 获取地图获客数据 |
| `/api/wechat/customers` | POST | 添加客户 |
| `/api/wechat/chat/list` | GET | 获取聚合聊天列表 |
| `/api/wechat/chat/:id/messages` | GET | 获取聊天记录 |
| `/api/wechat/chat/:id/messages` | POST | 发送消息 |
| `/api/wechat/tags` | GET | 获取标签列表 |
| `/api/wechat/tags` | POST | 创建标签 |
| `/api/wechat/tags/:id` | DELETE | 删除标签 |
| `/api/wechat/keyword-replies` | GET | 获取关键词回复列表 |
| `/api/wechat/keyword-replies` | POST | 创建关键词回复 |
| `/api/wechat/mass-send` | GET | 获取群发任务列表 |
| `/api/wechat/mass-send` | POST | 创建群发任务 |
| `/api/wechat/friends` | GET | 获取好友列表 |
| `/api/wechat/moments` | GET | 获取朋友圈任务列表 |
| `/api/wechat/moments` | POST | 创建朋友圈任务 |

## 🔧 核心功能说明

### 1. 地图获客 (MapCustomers.vue)
- ✅ 基于地理位置搜索附近潜在客户
- ✅ 支持距离筛选（500米、1公里、3公里、5公里）
- ✅ 客户详情查看和标签管理
- ✅ 一键添加好友功能

### 2. 聚合聊天 (AggregateChat.vue)
- ✅ 统一展示所有客户的聊天会话
- ✅ 实时消息收发
- ✅ 未读消息计数
- ✅ 快捷回复功能
- ✅ 支持发送文本、图片、文件

### 3. 标签管理 (Tags.vue)
- ✅ 创建/编辑/删除客户标签
- ✅ 标签颜色自定义
- ✅ 自动打标功能（基于关键词）
- ✅ 查看标签下所有成员
- ✅ 按标签批量群发消息

### 4. 关键词回复 (KeywordReply.vue)
- ✅ 设置关键词自动回复
- ✅ 支持精确匹配和模糊匹配
- ✅ 延迟回复功能（更真实）
- ✅ 支持多种回复类型（文本、图片、语音、视频）
- ✅ 使用次数统计

### 5. 精准群发 (MassSend.vue)
- ✅ 按标签选择目标客户
- ✅ 支持定时发送
- ✅ 智能间隔发送（避免封号）
- ✅ 支持图文消息
- ✅ 任务进度跟踪

### 6. 好友列表 (FriendList.vue)
- ✅ 查看所有微信好友
- ✅ 按标签筛选好友
- ✅ 好友搜索功能
- ✅ 好友详情查看
- ✅ 一键发起聊天

### 7. 朋友圈任务 (MomentsTask.vue)
- ✅ 创建朋友圈内容
- ✅ 支持定时发布
- ✅ 图片上传（最多9张）
- ✅ 可见范围设置
- ✅ 添加位置信息
- ✅ 朋友圈预览功能

## 🚀 如何使用

### 1. 启动后端服务
```bash
cd marketai-pro/src/backend
npm install
npm run dev
```

### 2. 启动前端服务
```bash
cd marketai-pro/src/frontend
npm install
npm run serve
```

### 3. 访问智能获客模块
- 打开浏览器访问 `http://localhost:8080`
- 登录后点击左侧菜单 "智能获客"
- 选择子功能：地图获客、聚合聊天、标签管理等
